## ⚙️ Techniques CSS utilisées

- **Flexbox** — navigation, boutons, carte profil, section contact
- **CSS Grid** — grille de compétences (`auto-fit, minmax`), grille de projets, formulaire deux colonnes
- **Variables CSS** — palette de couleurs centralisée via `:root`
- **Pseudo-éléments** `::before` / `::after` — effets décoratifs et barres colorées
- **Media Queries** — responsive à 900px, 768px et 480px
- **Transitions & Hover** — effets de survol sur cartes, liens et boutons
- **Dégradés CSS** — `linear-gradient` sur fonds, boutons et vignettes

## 📱 Responsive Design

Le site est adapté à tous les écrans :

| Écran | Comportement |
|---|---|
| Desktop (> 900px) | Mise en page en deux colonnes |
| Tablette (≤ 900px) | Colonnes empilées, centrage du contenu |
| Mobile (≤ 768px) | Formulaire en colonne unique |
| Petit mobile (≤ 480px) | Menu masqué, grille en une colonne |
